Lab 13 - Loops

Challenge

We had to use Javascript to use a "for loop" which goes through each number until 200, and if they're multiple of 3, 5, and 7 then they say Fizz! Buzz! and Boom! respectively. If there is a number that's both a multiple of 2 of the numbers, then it prints a combination of words.

Problems

My code wasn't printing in the output in the beginning, and I realized it was because I had included the output into the loop, so it wasn't outside. After I moved the .html in the JS outside, it printed.

It's very reminiscent of what I remember learning in Python, as these looping functions were the only one I remember being able to figure out on my own.
